## Tarefas feitas

Obrigatoriamente para o desenvolvimento do back-end utilizar o framework Laravel.
Obrigatoriamente a API deve estar nos padrões RESTful.
Desenvolvido uma listagem de pacientes com busca, do qual deve-se permitir a adição, edição, visualização e exclusão de cada um dos pacientes.

Utilizado para banco de dados PostgreSQL e Redis (Cache e Queue).
Utilizado migration, factory, faker e seeder.
Criado um endpoint para listagem onde seja possível consultar pacientes pelo nome ou CPF.
Criado um endpoint para obter os dados de um único pacientes (paciente e seu endereço).
Criado endpoints de cadastro e atualização de paciente
Criado um endpoint para excluir um paciente (paciente e seu endereço).
Criado um endpoint para consulta de CEP que implemente a API do ViaCEP e faça cache (Redis) dos dados para futuras consultas.
Criado um endpoint que faça importação de dados (pacientes) via arquivo .csv e seja processada em queue assincronamente.
